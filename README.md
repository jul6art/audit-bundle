jul6art/audit-bundle
====================
Symfony audit bundle
--------------------

> :warning: Work in progress so keep calm. The good news: this is maintained!

Requirements
------------

* **php ^7.4 || ^8.0**
* **symfony ^4.4 || ^5.0**

Installation
------------

```console
composer require jul6art/audit-bundle
```
